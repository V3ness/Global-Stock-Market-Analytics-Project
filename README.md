# Global-Stock-Market-Analytics-Project

## Problem Statement
Dataset used in this project: https://www.kaggle.com/datasets/jacksaleeby/global-stock-market-indices-4-6m-rows

This Databricks project transforms a 4.6 million–row dataset into meaningful insights using PySpark, SQL, and Databricks dashboards to enable exploration of global index behavior and performance.
Global financial markets move quickly and are influenced by economic, geopolitical, and regional events. Organizations must continuously evaluate global stock indices to identify market trends and volatility. This Databricks project transforms a 4.6 million–row dataset into meaningful insights using PySpark, SQL, and Databricks dashboards to enable exploration of global index behavior and performance.
